第一步 进入机器

第二步 进入相应目录 并解压

我这里以zip压缩包为例，因为Ubuntu自带了unzip所以直接使用就好。

第三步 其他压缩包

如果是RAR/tar.gz/gz这些压缩包的用户，命令如下

gz解压

gzip -d file.gz

tar.gz解压

tar -xzvf myetc.tar.gz

rar解压

apt-get update
apt-get install rar unrar
unrar  x  test.rar

因为矩池云里面是一个Ubuntu系统，基本上Ubuntu的命令都是适用的。
